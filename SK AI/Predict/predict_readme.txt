[predict_service.csv]

ㅇ 가입일이 3/13부터 3/15까지인 서비스들 중 해지 예측일에 유지되고 있는 서비스 데이터임
      - 가입일 : registerdate
      - 해지 예측일 : registerdate + 3 weeks
      - 해지 예측일 당시 유지 여부 : enddate > registerdate + 3 weeks
ㅇ 해지 예측일에 유지되고 있는 서비스를 가입한 고객 중 
    ① 해당 서비스 이전에 가입한 서비스 이력이 있는 경우 이 데이터도 포함되어 있음
    ② 하나 이상의 서비스를 유지하고 있을 수도 있음
        [참고] 상품(이용권) 중에는 중복 가입이 가능한 상품이 있고, 그렇지 않은 상품이 있음 (단, 제공된 상품 정보에는 코드화되어 있지 않음)


[predict_bookmark.csv]

ㅇ 해지 예측 대상 서비스 가입 고객의 시청 데이터로 해지 예측전일까지만 포함되어 있음
      - 예측 전일 : registerdate + 3 weeks - 1 day
